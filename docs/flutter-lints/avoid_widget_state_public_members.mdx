# avoid_widget_state_public_members

- Severity: info ℹ️

## Details

**AVOID** declaring public members in widget state classes.

```dart title="Bad"
class _MyWidgetState extends State<MyWidget> {
  int counter = 0;

  void increment() {
    setState(() {
      counter++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return FilledButton(
      onPressed: increment,
      child: Text('$counter'),
    );
  }
}
```

```dart title="Good"
class _MyWidgetState extends State<MyWidget> {
  int _counter = 0;

  void _increment() {
    setState(() {
      _counter++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return FilledButton(
      onPressed: _increment,
      child: Text('$_counter'),
    );
  }
}
```

## Usage

To enable the `avoid_widget_state_public_members` rule, add `avoid_widget_state_public_members` under custom_lint > rules in your `analysis_options.yaml` file:

```yaml
custom_lint:
  rules:
    - avoid_widget_state_public_members
```
