# avoid_inverted_boolean_expressions

Unnecessary inverted boolean expression should be avoided since it decreases code readability.

- Severity: info
- Quick fix: âœ“

Bad

```dart
if (!(number > 0)) {}
final text = !(number == 0) ? 'Not zero' : 'Zero';
```

Good

```dart
if (number <= 0) {}
final text = number != 0 ? 'Not zero' : 'Zero';
```
