# prefer_iterable_every

- Severity: info ℹ️
- Quick fix: ✅

## Details

**DO** use `iterable.every` instead of `iterable.where().isEmpty`.

```dart title="Bad"
const numbers = [1, 2, 3];
final areAllPositive = numbers.where((number) => number < 0).isEmpty;
```

```dart title="Good"
const numbers = [1, 2, 3];
final areAllPositive = numbers.every((number) => number > 0);
```

## Usage

To enable the `prefer_iterable_every` rule, add `prefer_iterable_every` under custom_lint > rules in your `analysis_options.yaml` file:

```yaml
custom_lint:
  rules:
    - prefer_iterable_every
```
