# What is Pyramid Lint?

Pyramid Lint is a linting tool for Dart and Flutter projects. It offers a set of additional lints and quick fixes to help developers identify issues with their Dart code, suggests potential fixes, and enforces a consistent coding style.

<Info> Pyramid Lint is built with [custom_lint][custom_lint]. </Info>

## Features

- Lints: Identifies potential issues in your code.
- Quick fixes: Suggests potential fixes for the issues identified by lints.
- Assists: Provides refactoring and code completion for common tasks.
